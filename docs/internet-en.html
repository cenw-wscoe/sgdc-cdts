Implementing the GCWeb theme using the CDTS framework in your application

CDTS gives users the flexibility of implementing the GCWeb theme in two different ways: the first method uses document.write(). The newly introduced second method (optional) avoids the need of document.write(), having to call multiple CDTS functions and optionally the need for inline scripts altogether.

The documentation for the first method the uses document.write() can be found here.

The documentation for the second method that does not use document.write() can be found here.

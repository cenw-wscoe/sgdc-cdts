Intégrité des sous-ressources (ISR) pour le SGDC

Slogan

Note

L'intégrité des sous-ressources est obtenue en attachant l'attribut d'intégrité aux éléments HTML qui téléchargent des ressources externes (lien, img, script, piste, iframe, vidéo, audio, etc.). Cet attribut à une valeur qui est une combinaison de la valeur de hachage de la ressource, de l'algorithme de hachage et du type MIME du fichier.

Le but de cette page est de répertorier les valeurs ISR pour les fichiers de la couche de présentation jQuery SGDC. Le SGDC a inclus le ISR pour les fichiers jQuery JS à partir de la v4.0.23 du BOEW/SGDC.

Vous ne savez pas ce qu'est l'ISR ? Lisez cet article(en anglais seulement) !
